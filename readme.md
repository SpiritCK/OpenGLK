# OpenGLK
## Libraries required
- cmake
- make
- g++
- libx11-dev
- libxi-dev
- libgl1-mesa-dev
- libglu1-mesa-dev
- libxrandr-dev
- libxext-dev
- libxcursor-dev
- libxinerama-dev
- libxi-dev
- libglew-dev
- libglfw3-dev

For Linux user, just use
`sudo apt-get install cmake make g++ libx11-dev libxi-dev libgl1-mesa-dev libglu1-mesa-dev libxrandr-dev libxext-dev libxcursor-dev libxinerama-dev libxi-dev libglew-dev libglfw3-dev`

## Compilation
`./compile.sh`

## How To Run
`./bin/[program_name]`
